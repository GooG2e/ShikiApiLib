# ShikiApi(String, String, Int32) - конструктор


Инициализирует нового пользователя, явно задавая значения токена и ID пользователя.

**Пространство имён:**&nbsp;<a target="_blank" href="N_ShikiApiLib.md">ShikiApiLib</a>

## Синтаксис

**C#**<br />
``` C#
public ShikiApi(
	string nickname,
	string access_token,
	int current_user_id
)
```
